
## 简介
头文件 `algorithm` 里的函数实现也很简单，除了极个别的几个外，其它的直接遍历即可

详细说明，见函数实现及其注释

## 问题
感觉，很多函数，例如，std::sort，可以只要求前向迭代器，
内部对，前向、双向和随机迭代器重载即可
