
## 简介
头文件 `cctype` 里的所有函数实现都很简单，需要注意的是：

* 并不是所有的编码系统都是 `ASCII` 或 兼容 `ASCII` 的
* 参数 `ch` 必须是 `EOF` 或 可表示为 `unsigned char`，否则，是未定义的行为
