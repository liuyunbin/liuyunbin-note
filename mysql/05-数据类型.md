
## 常用类型
```
INT --------------- 整形 ---------------------- 4     个字节
BIGINT UNSIGNED --- 大整形 -------------------- 8     个字节
DECIMAL(M,D) ------ 定点数 -------------------- M + 2 个字节
DATETIME ---------- 日期和时间 ---------------- 8     个字节
TIMESTAMP --------- 时间戳 -------------------- 4     个字节
CHAR(M) ----------- 固定长度, M 为字符数 ------
VARCHAR(M) -------- 可变长度, M 为字符数 ------
ENUM -------------- 枚举 ---------------------- 1 或 2 个字节, 只能取一个值
SET --------------- 集合 ---------------------- 1 到 8 个字节, 可以取 0 个或多个
BINARY(M) --------- 固定长度, 二进制字符串 ---- M   个字节
VARBINARY(M) ------ 可变长度, 二进制字符串 ---- M+1 个字节
JSON -------------- JSON (使用 -> 或 ->> 取成员值)
```
